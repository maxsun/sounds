
This repo contains scripts & resources for making and analyzing sounds.

You will need:

1. [PyAudio](https://people.csail.mit.edu/hubert/pyaudio/#downloads), which (on debian) may require you to run:
```
sudo apt-get install libasound-dev portaudio19-dev libportaudio2 libportaudiocpp0
sudo apt-get install ffmpeg libav-tools
```
